Ticket Lifecycle (Notes)

  1. User raises ticket → System assigns via rules
  2. Team Lead reviews → Assign technician
  3. Technician updates thread & uploads proof
  4. User verifies and closes / reopens if needed
  5. Admin monitors SLAs and reassigns as needed
